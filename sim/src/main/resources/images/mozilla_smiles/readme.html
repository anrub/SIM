Mozilla Thunderbird Smilies


These are the smiles from the Mozilla Thunderbird email program.

Demo

                               
(Note: if you are using Internet Explorer, the gray background is not part of the smilie--they are transparent.)

Installation

Simply upload all smilies and the mozilla_smilies.pak file to your_forums_directory/images/smilies.
Now go to your Administration Panel->General Admin->Smilies->Import Smilie Pack->Select mozilla_smilies.pak and click Import Smilies.
(Note: make sure that "Delete existing smileys before import" is UNCHECKED and "What should be done in case of conflicts" is set to "Keep Existing Smiley".)
Now you're done!
